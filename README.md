## ğŸ²&nbsp; About

This Android application replicates the interface seen in the Netflix series Alice in Borderland. Its initial purpose is to present game-related information for the players, such as the gameâ€™s name and difficulty level, as illustrated in the screenshots below.

<p align="center" href="#title">

Game selection and management are performed by the GameMaster using the <a href="https://github.com/edvaldinhs/bordera">bordera</a> application. Communication is implemented through periodic state polling from a custom API, with the GameMaster issuing pings/signals to modify the current state and game_id.
